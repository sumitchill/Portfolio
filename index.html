Oled Screen BurnIn Prevention

I built a device that automatically turn on/off my OLED monitor screen with a motion sensor. I use this OLED screen as a computer monitor , so this helps protect the OLED againts burn in when i'm away from the display. This project is designed to automatically turn on/off an OLED monitor or TV screen using an ESP32 microcontroller and a motion sensor. The primary goal is to protect the OLED display from burn-in when the user is away from the display.
